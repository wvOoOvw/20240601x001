## Component Common Property 组件通用属性

- **isolated**
  - **boolean**
  - **默认不开启**
  - **开启后：组件自存储一份画布图形/样式状态进而不会影响到子组件**

- **beginPath**
  - **boolean**
  - **默认不开启**
  - **开启后：组件绘制前清空先前绘制的几何路径**

- **fill**
  - **boolean**
  - **默认不开启**
  - **开启后：使用填充模式绘制**

- **fillStyle**
  - **string**
  - **设置填充模式的颜色**
