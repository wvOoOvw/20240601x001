## Component Common Property 组件通用属性

- **isolated**
  - **boolean**
  - **默认不开启：组件的图形/样式状态会传播给子组件**
  - **开启后：组件自存储一份画布图形/样式状态进而不会影响到子组件**

- **fill**
  - **boolean**
  - **开启后：使用填充模式绘制**

- **fillStyle**
  - **string**
  - **设置填充模式的颜色**

- **x**
  - **number**
  - **矩形左上角x轴坐标位置**

- **y**
  - **number**
  - **矩形左上角y轴坐标位置**
